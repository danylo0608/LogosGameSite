# LogosGame Static Site (LG_SITE)

Це статична версія сайту LogosGame, яка працює на чистому HTML/CSS/JS та використовує бібліотеку `sql.js` для роботи з базою даних SQLite безпосередньо у браузері.

## Як запустити
Просто відкрийте файл `index.html` у будь-якому сучасному браузері. Сайт працюватиме навіть без локального сервера.

## Як оновити базу даних
Оскільки браузери блокують читання локальних файлів з міркувань безпеки (CORS), база даних автоматично конвертується у JS-файл.

Щоб оновити дані:
1. Виконайте скрипт `update_static_db.py` у кореневій папці проекту:
   ```bash
   python update_static_db.py
   ```
2. Файл `LG_SITE/js/db_bin.js` буде автоматично оновлено актуальними даними з вашої основної бази `db.sqlite3`.

## Особливості
- **Без сервера**: Працює офлайн та при прямому відкритті файлів.
- **SQL у браузері**: Використовує WebAssembly версію SQLite.
- **Динамічність**: Фільтри та пошук працюють миттєво.
